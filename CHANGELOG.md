# Changelog

All notable changes to QR Studio will be documented in this file.

## [1.0.0] - 2024-03-24

### Added
- Initial release of QR Studio
- Support for 7 QR types: URL, Text, Email, Phone, WiFi, vCard, WhatsApp
- Real-time color preview when changing colors
- AJAX form submission (no page refresh)
- Logo overlay support
- Multiple QR customization options
- Modern UI with Tailwind CSS
- Dark mode support
- Toast notifications
- Drag & drop file upload
- Local storage for QR persistence
- REST API endpoints for QR generation
- Comprehensive documentation

### Features
- Professional, aesthetic UI design
- Mobile responsive layout
- Loading spinners and feedback
- Error handling and validation
- Download generated QR codes
- Batch generation support (future)
- Analytics tracking (future)

## [0.9.0] - 2024-03-24

### Added
- Beta version with core QR generation
- Basic Flask application structure
- HTML template with form
- QR code generation with Pillow
- Color customization (hex to RGB)
- Logo handling with circular background

### Changed
- Improved error correction levels
- Enhanced logo positioning
- Better color contrast handling

### Fixed
- Page refresh on form submission
- JavaScript scope issues
- Color parsing from hex strings
- AJAX event listener binding
